# Análisis técnico de CockroachDB

Investigación individual desarrollada para la asignatura **Aplicaciones Distribuidas**.

El documento analiza a **CockroachDB** como tecnología de base de datos distribuida, considerando su arquitectura, distribución de datos, modelo de consistencia según el teorema CAP, un caso de uso real y su posible aplicación en el proyecto **TiendaTech**.

## Autor

**Jeremy Ruperto Gaibor Rodríguez**

## Contenido de la investigación

El documento incluye:

- Distribución de datos mediante rangos y réplicas.
- Uso del protocolo de consenso Raft.
- Clasificación de CockroachDB como sistema CP según CAP.
- Transacciones ACID y aislamiento serializable.
- Caso real de Hard Rock Digital.
- Análisis de pertinencia para el proyecto TiendaTech.
- Referencias bibliográficas en formato IEEE.
